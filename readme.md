## Installation

*Installing with [Clib](https://github.com/clibs/clib)*

```sh
$ clib install abranhe/os
```

## Usage

If you run:

```
$ os --help
```

You will get:

```
  Usage: os [options]

  Options:

    -h, --help      output help information
    -v, --version   output os version

  Examples:

    $ os
    macOS
```
